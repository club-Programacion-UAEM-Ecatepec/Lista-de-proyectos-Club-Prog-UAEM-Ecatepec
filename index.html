Bienvenido a la lista de proyectos de nuestro club

Esta es una página web y repositorio con el listado de los proyectos que estan siendo desarollados por los integrantes del Club de Programación y cualquier miembro del club puede buscar o registrar un proyecto

Busca proyectos con palabras clave en buscador o registra un proyecto propio para tener registro o buscar colaboradores

Formato para agregar un proyecto

Puedes agregar tu proyecto a la lista si cumples con los siguientes requisitos:

  1. Eres parte del Club de Programación UAEM Ecatepec
  2. Tienes registrada una cuenta de GitHub
  3. El proyecto que vas a registrar esta en tu GitHub
  4. Sigues el formato adecuado para agregar tu proyecto a la lista

Si cumples los puntos 1, 2 y 3, sigue las instrucciones del siguiente enlace Formato para agregar mi proyecto para agregar tu proyecto a la lista correctamente
